# さとう数理塾 プロフィール

## 基本情報
- 塾名: さとう数理塾
- 対象エリア: 沼津・三島エリア
- 対象学年: 小・中・高校生（高校生は理系志望中心）
- 所在地: 〒410-0052 静岡県沼津市沢田町1-3 牧原ビル102

## 強み（数学教育）
- 数学教育が得意: 数学教育専攻＋数学の教員免許（中・高）を持つ塾長が、数学のつまずきを構造から整えて伸ばす
- 数学を軸に全体最適: 数学を起点に、必要科目へ波及させて受験戦略を組む

## 高校生向けの印象（理系専門）
- 理系大学受験の専門塾: 数学を中心に、英語・物理・化学までまとめてサポート（共通テスト＋国公立なら二次まで見据える）

## 開校・通い方（自由通塾制）
- 開校日: 月〜金
- 開校時間: 18:50〜21:50
- 休み: 土日・お盆・年末年始
- 通塾ルール: 曜日固定なし／来たい日に来てOK
- 連絡: 遅刻・欠席の連絡不要
- 滞在時間: 最大180分の間で滞在自由
- 回数: 週何回来ても月謝は同じ（回数制限なし）／テスト前だけ増やすのもOK

## 学び方（AI教材×塾長サポート）
- 土台: AI教材「atama+（アタマプラス）」の演習サイクル
  - 特徴: 新しい単元の学習に必要な復習単元を自動で提案（例: 一次関数を学ぶ前に、必要な方程式・比例の単元を自動特定）
  - 小学生: 算数・国語・英語・理科・社会
  - 中学生: 数学・英語・理科・社会・国語（国文法・漢字）
  - 高校生: 数学・英語・物理・化学・生物・地理・日本史・世界史・地歴総合・情報・古典
  - ※2026年3月時点の対応教科
- 学習の型: 解説は短く、演習は長く（説明→即演習／つまずいたら短い解説→演習へ戻る）
- 回し方: 演習→つまずき→解説→演習 を確実に回す
- 確認: 節目ごとに確認テスト／できなかった所だけ戻ってやり直し（「解けたつもり」を残さない）
- サポート: 塾長が学習状況を見ながら声かけ／つまずき解消をその場で支援
- 教材運用: タブレットだけで完結せず、学校教材・過去問など紙教材も併用
- 質問: 演習中いつでもOK（止まっている所を一緒にほどく）
- 学校対応: 学校ワークやテスト範囲は、学校の進度に合わせて対応

## 設備・サポート
- Chromebook貸出: 塾内で使用するChromebookを無料で貸し出し（端末の持ち込み不要）
- 三者面談: 保護者・生徒・塾長の三者面談を実施（学習状況の共有・進路相談）
- AI質問チャットアプリ（塾生限定プレミアム機能）
  - スマホのカメラで問題を撮影するだけでAIが即座に解析・解説（数式入力不要）
  - 数学だけでなく英語・物理・化学など複数教科の質問に対応
  - 難関大学の入試問題も解ける高い解析力
  - 答えを教えるだけでなく「ヒント」で思考力を促し、「類題」で定着を図る
  - 質問した問題は自動でリスト化、AIが最適なタイミングで復習を促す（自動復習機能）
  - 24時間利用可能（塾にいない時間でも学習サポート）
  - 無料AIアプリとの差別化: 解説の深さ・類題生成・復習管理まで一貫サポート

- 担当: 塾長（佐藤）が最初から最後まで一貫して担当（途中で担当変更なし）
- 学歴: 横浜国立大学 教育人間科学部 数学教育専攻 卒
