# marketing plugin

さとう数理塾専用Webマーケティングコンサルタント。

## 機能

- GA4・Search Console・Firecrawlのデータに基づく現状分析
- SEO施策（meta最適化、構造化データ、学年別LP作成）
- コンテンツ企画・作成（ブランドボイス準拠）
- GBP最適化、Instagram投稿企画
- CVR改善、競合分析
- 定期レポート（週次/月次）

## 依存MCP

- `analytics-mcp` — GA4 Data API
- `mcp-gsc` — Search Console API
- `firecrawl` — 競合サイトスクレイピング

## 依存ファイル（プロジェクト側）

- `.agents/product-marketing-context.md` — PMC（ビジネス実態・ブランドボイス）
- `memory/marketing-plan.md` — 全体計画
- `memory/MARKETING.md` — 施策ログ・仮説ログ
- `memory/marketing-research.md` — 調査知見
- `memory/local-marketing-knowledge.md` — 塾ローカルマーケティング知識ベース
