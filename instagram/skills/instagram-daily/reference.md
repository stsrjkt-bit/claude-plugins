# instagram-daily トーン定義

## 基本トーン: 1970年代教育実践記録風

大村はま・斎藤喜博的な「教室の息づかい」を文章にする。
子どもの小さな変化を丁寧に拾い、先生の素朴な感想として綴る。

### 書き方の原則

1. **子どもの変化を拾う**: 「できた」ではなく「目の色が変わった」「手が止まらなくなった」「ノートの字が変わった」
2. **数値は使わない（or 最小限）**: 「55分で3単元合格」→「算数のモヤモヤを解消して、そのまま中1に入れた」。どうしても数字を出すなら「1時間ほど」程度にぼかす
3. **先生の素朴な感想で締める**: 「笑顔で帰ってくれた」「ちょっと目の色が変わった気がする」「次来るのが楽しみだ」
4. **箇条書き禁止**: 短い文章を3-4段落で。段落の切れ目で呼吸を入れる
5. **教室の空気を描写する**: 「鉛筆の音だけが聞こえる時間があった」「ふっと肩の力が抜けた瞬間が見えた」

### 禁止ワード・表現

以下は絶対に使わない:

- **SNSマーケ用語**: 「伴走」「サポート」「寄り添う」「見守る」「応援」「全力で」
- **教育業界バズワード**: 「個別最適化」「主体的な学び」「自走」「学びの伴走者」「成長をサポート」
- **煽り表現**: 「今がチャンス」「差がつく」「このままで大丈夫？」「まだ間に合う」
- **自塾アピール**: 「当塾では」「さとう数理塾なら」「AI教材だから」
- **過度な感嘆**: 「すごい！」「やった！」「感動しました！」（「！」は1投稿1-2回まで）

### 良い例

```
体験で来てくれた小6の子。算数がちょっと苦手で、分数の計算あたりから怪しかった。

ひとつずつ穴を埋めていったら、思ったより早く追いついて、そのまま中1の正負の数に入れた。
本人も「え、もう中学の？」という顔をしていた。

帰り際、お母さんに「楽しかった」と言っていたのが聞こえた。
こういう一言が、いちばん嬉しい。
```

### 悪い例

```
本日、体験授業にお越しいただいた小6のお子さまをサポートさせていただきました！

AI教材を活用し、個別最適化された学習プランで算数の弱点を効率的に克服。
わずか55分で3単元をクリアし、中1範囲まで到達しました！

さとう数理塾では、一人ひとりに寄り添った伴走型の指導を大切にしています。
体験授業、随時受付中です！
```

## instagram/reference.md との関係

instagram/reference.md のトーン指定（「保護者の友達に話す距離感」「生徒さん表記」「締めは具体的な観察」等）も併せて遵守する。
本ファイルのトーン定義はそれをさらに「教育実践記録」方向に深めたもの。両方を守ること。

## 画像テキスト（headline / subhead / body）

- headline はキャッチーにしつつ、煽りにならない温度感。「問いかけ型」「体験談型」が相性が良い
- body は画像テキストだけで「何があったか」が伝わるように。キャプションは読まれない前提
- 文字数はinstagram/reference.md のテキスト制約に従う
