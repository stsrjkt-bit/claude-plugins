# Instagram投稿 共通参照資料

## 塾プロフィール

```
塾名：さとう数理塾（沼津・三島エリア）
特徴：AI教材×塾長の個別サポート、自由通塾制（定額で通い放題）
対象：小中高生（高校生は理系中心）
運営：塾長（佐藤）が一人で担当
```

## 画像自立の原則（★最重要★）

キャプションは読まれない前提で、画像テキストだけで内容が伝わるようにする。
「何の話か」「いつの話か」が画像だけで明確にわかること。

## トーン指定（★最重要★）

### 基本姿勢: 1970年代教育実践記録風
大村はま・斎藤喜博的な「教室の息づかい」を文章にする。
子どもの小さな変化を丁寧に拾い、先生の素朴な感想として綴る。

### 距離感・文体
- 「保護者の友達に話す」くらいの距離感で書く
- 箇条書き・リストは使わない（短い文章を3〜4段落で。段落の切れ目で呼吸を入れる）
- 最後に他学年や他サービスへの宣伝を入れない
- 「！」は1つの投稿で1〜2回まで
- 締めの文は先生の素朴な感想で静かに終わる（抽象的にしない）
  - NG: 「穏やかに送り出せそうです」
  - OK: 「いい表情で帰っていったので、大丈夫だと思います」

### 書き方の原則
1. **子どもの変化を拾う**: 「できた」ではなく「目の色が変わった」「手が止まらなくなった」「ノートの字が変わった」
2. **数値は使わない（or 最小限）**: 「55分で3単元合格」→「算数のモヤモヤを解消して、そのまま中1に入れた」。どうしても数字を出すなら「1時間ほど」程度にぼかす
3. **先生の素朴な感想で締める**: 「笑顔で帰ってくれた」「ちょっと目の色が変わった気がする」「次来るのが楽しみだ」
4. **教室の空気を描写する**: 「鉛筆の音だけが聞こえる時間があった」「ふっと肩の力が抜けた瞬間が見えた」

### 呼称ルール
- 「生徒」は必ず「生徒さん」と書く（呼び捨て禁止）
- 教室描写の文脈では「小6の生徒さん」「中2の生徒さん」と書く
- 「子」「お子さま」「生徒」（敬称なし）は使わない

### 禁止ワード・表現
以下は絶対に使わない:
- **硬い表現**: 「プロとして」「役割」
- **SNSマーケ用語**: 「伴走」「サポート」「寄り添う」「見守る」「応援」「全力で」
- **教育業界バズワード**: 「個別最適化」「主体的な学び」「自走」「学びの伴走者」「成長をサポート」
- **煽り表現**: 「今がチャンス」「差がつく」「このままで大丈夫？」「まだ間に合う」
- **自塾アピール**: 「当塾では」「さとう数理塾なら」「AI教材だから」
- **過度な感嘆**: 「すごい！」「やった！」「感動しました！」

### 良い例

```
体験で来てくれた小6の生徒さん。算数がちょっと苦手で、分数の計算あたりから怪しかった。

ひとつずつ穴を埋めていったら、思ったより早く追いついて、そのまま中1の正負の数に入れた。
本人も「え、もう中学の？」という顔をしていた。

帰り際、お母さんに「楽しかった」と言っていたのが聞こえた。
こういう一言が、いちばん嬉しい。
```

### 悪い例

```
本日、体験授業にお越しいただいた小6のお子さまをサポートさせていただきました！

AI教材を活用し、個別最適化された学習プランで算数の弱点を効率的に克服。
わずか55分で3単元をクリアし、中1範囲まで到達しました！

さとう数理塾では、一人ひとりに寄り添った伴走型の指導を大切にしています。
体験授業、随時受付中です！
```

## テキスト制約（目安）

文字数は目安。伝わりやすさを優先し、必要なら超えてよい。

- **headline**: 8文字程度（伝わるなら10文字超えてもOK）
- **subhead**: 20文字程度
- **body**: 40文字程度、改行で2〜3行に
- **caption**: 200〜300文字

## キャプション構成

1行目：時期に関する共感や挨拶
2段落：今日の出来事の報告（塾長の入力をベースに、子どもの小さな変化を拾う）
3段落：なぜこの時期にそれが大事か（リサーチ情報を活用して文脈と結びつける）
4段落：先生の素朴な感想で静かに締める
最後：ハッシュタグ（#さとう数理塾 #沼津 など5個程度）
※商圏が半径500m程度のため、三島関連のタグは不要

## 出力JSON形式

```json
{
  "imageText": {
    "headline": "1行目タイトル（8文字以内）",
    "subhead": "2行目タイトル（10文字以内）",
    "body": "本文（40文字以内、改行2〜3回）"
  },
  "caption": "キャプション本文（200〜300文字、段落分け、最後にハッシュタグ）"
}
```
