# UI Design to React プラグイン

Stitch MCP + HTMLモック + Figmaキャプチャを組み合わせたUI制作ワークフロー。
デザインモック → Stitch生成 → 比較 → Figma統合 → React実装 の5フェーズ。

## 前提条件

- Stitch MCP（`TEXT_TO_UI_PRO` プロジェクト）
- Playwright MCP（スクショ・プレビュー用）
- Figma MCP（デザイン統合用）
- Doppler（シークレット管理）
