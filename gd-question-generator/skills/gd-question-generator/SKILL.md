# /gd-question-generator — GD予想問題生成スキル

滋賀県立大学 工学部 材料化学科 後期試験のグループディスカッション（4〜5人）予想問題を、高校化学の教科書を起点に網羅的に生成する。

---

## Hard Gates（前提条件）

- このスキルは **問題生成のみ** を行う。模範解答・ディスカッションの進め方は別スキルの責務
- 生成する問題の温度感は **2024-2025年レベル** に固定する。これが最重要制約
- すべてのテーマは高校化学の教科書に登場する概念・物質と紐づくこと

---

## 過去問分析（出題パターンの根拠）

### 実際の過去問

| 年度 | テーマ | 評価 |
|------|--------|------|
| 2025 | Li, Ca, Si, Cu のいずれか一つが地球から枯渇したら、生活にどのような支障が生じるか。元素を一つ選択し、具体例を挙げて説明せよ。 | ⭕ 最高傑作 |
| 2024 | 化学関連の仕事に就くとして、大学で「化学」以外に学ぶ科目として「数学」と「物理」のどちらがより重要か。 | ⭕ まともな議論に |
| 2023 | 空飛ぶクルマが実用化され始めている。日常的に使用することを日本で認めてもいいか。 | ❌ 化学関係薄い |
| 2022 | LED信号機への切替で消費電力が減った。同じ電気代で信号機を増やすか、LED化で電気代を安くするか。 | ❌ 専門的すぎ |
| 2021 | 翻訳ソフト・通訳機器の発達で外国語スキルの意味が薄れている。英語教育を継続すべきか。 | ❌ 化学ゼロ |

### 出題者の試行錯誤（分析）

```
2021 英語教育      → 化学関係なし。ぼんやりした議論に
2022 LED信号       → 急に化学寄りに。受験生に専門知識なく議論低調
2023 空飛ぶクルマ   → 未来技術だが法律の話に。化学関係薄い
2024 数学vs物理     → 化学の教科書知識だけで語れる → まともな議論に！
2025 元素×社会影響  → 教科書の元素 × 「なくなったら？」→ 一番うまくいった
```

### 確定した出題パターン（2024-2025で収束）

> **高校化学の教科書で見かけるもの × 「それが社会でどう使われている？」「どっちが大事？」「なくなったら？」**

- 化学の専門知識そのものではなく「化学×社会」が問われる
- 高校生が背景知識を持っているレベルのテーマ
- 「選択肢を提示 → 1つ選ばせる → 根拠を述べさせる」形式
- 4〜5人で10分程度議論が成立する粒度

## 温度感のガイドライン（最重要）

### ❌ 温度が高すぎる（大学レベル・出ない）
- 「ペロブスカイト系酸化物触媒のOER活性について議論せよ」
- 「LaNi₅の水素吸蔵メカニズムの課題を述べよ」
- 「YSZ電解質を用いたSOECの材料設計について」

### ❌ 温度が低すぎる（化学関係ない・2021-2023に逆戻り）
- 「SNSは学生に良い影響を与えるか悪い影響を与えるか」
- 「リモートワークは今後も続けるべきか」
- 「宇宙開発に予算を使うべきか」

### ⭕ ちょうどいい温度感（2024-2025型）
- 「Li, Ca, Si, Cu のうち枯渇したら最も困る元素は？」→ 教科書で見る元素 × 身近な影響
- 「化学に数学と物理どっちが大事？」→ 化学を学ぶ姿勢を問う
- 「水素エネルギーで製造・貯蔵・利用のどれが最大の課題？」→ 教科書の反応 × エネルギー問題
- 「生分解性プラスチック vs リサイクル、どっちが有効？」→ 高分子の知識 × 環境問題

## 問いかけパターン（テンプレート）

過去問から抽出した問いかけの型。この5パターンを使い回す。

### パターン1：枯渇型（2025年型）
```
「A, B, C, D のいずれか一つが〇〇したら、△△にどのような影響が生じるか。
一つ選び、具体例を挙げて説明せよ」
```
→ 元素・資源テーマと相性が良い

### パターン2：二者択一型（2024年型）
```
「〇〇と△△のどちらがより□□か。一方を選び、理由を述べよ」
```
→ 方法論比較、技術比較テーマと相性が良い

### パターン3：三者択一型
```
「A, B, C のうち、最も□□なのはどれか。一つ選び、具体的な根拠を挙げて論じよ」
```
→ プロセスの段階比較、複数要因比較テーマと相性が良い

### パターン4：選択＋将来展望型
```
「A, B, C, D のいずれか一つを選び、〇〇の観点から今後どのような役割を果たしうるか。
現在の用途と比較しながら具体的に論じよ」
```
→ 元素・材料テーマの将来展望と相性が良い

### パターン5：是非判断型（2023年型）
```
「〇〇は△△にとってプラスかマイナスか。自分の考えを述べよ」
```
→ 技術の社会的影響テーマと相性が良い（ただしこの型は2024以降使われていない）

## 生成ルール

1. **教科書の単元がベース**：全てのテーマは高校化学の教科書に登場する概念・物質と紐づく
2. **温度感は2024-2025年レベル**：高校生が10分で議論できるレベル
3. **立場が複数成立する**：正解が一つに決まらないこと
4. **4〜5人で議論可能**：広すぎず狭すぎないテーマ設定
5. **問いかけは5パターンのいずれかを使用**
6. **化学×社会の接点を問う**：純粋な化学知識の問題にしない
7. **カテゴリA〜Hを網羅**：特定分野に偏らない

## ワークフロー

### Phase 0: データ読み込み

1. `data/gd-themes.ts` を読み込み、既存のテーマ定義を確認する
2. カテゴリごとのテーマ数を集計し、カバレッジを把握する

### Phase 1: テーマ生成

ユーザーのリクエストに応じて、以下のいずれかを実行:

- **全カテゴリ生成**: カテゴリA〜Hの全テーマを一括生成
- **単一カテゴリ生成**: 指定カテゴリのテーマのみ生成
- **追加テーマ生成**: 既存テーマに追加する新テーマを生成

**生成手順:**
1. 教科書単元マッピングから対象テーマを選定
2. 適切な問いかけパターン（P1〜P5）を割り当て
3. 温度感ガイドラインに照らしてセルフチェック
4. 出力フォーマットに従ってYAMLブロックを生成

### Phase 2: 品質チェック

生成したテーマについて以下を確認:

- [ ] 温度感は2024-2025年レベルか（高すぎない？低すぎない？）
- [ ] 正解が一つに決まらないか（議論が成立するか）
- [ ] 4〜5人で10分議論できる粒度か
- [ ] 教科書の単元と紐づいているか
- [ ] カテゴリが偏っていないか
- [ ] 既存テーマと重複していないか

### Phase 3: 出力

結果をユーザーに提示する。以下のフォーマットを使用:

```yaml
id: "b1"
name: "水素エネルギーの課題"
emoji: "💧"
categoryId: "energy"
year: 2026
textbookUnit: "水の電気分解、燃料電池"
questionPattern: "pattern3"
question: "水素エネルギーの普及において、製造・貯蔵・利用のうち、最も大きな課題はどれか。一つ選び、具体的な根拠を挙げて論じよ。"
choices: ["製造", "貯蔵", "利用"]
```

ユーザーが希望する場合、`data/gd-themes.ts` のTypeScript定数ファイルとしても出力する。
